使用pycharm（专业版）连接docker容器

安装pycharm专业版之后有很多实用的功能，其中使用pycahrm连接docker容器，使我们不必来回传代码，实现本地文件与容器文件同步，开发效率杠杠的

pycharm专业版、Miniconda Ubuntu安装以前写过不再赘述

一、远端服务器部分

1. 准备基础镜像

拉取镜像

2. 启动容器

  • –name （给容器命名）：防止服务器中启动的镜像太多，找不到了
  • -v（映射文件目录） : 服务器文件目录 : 容器目录
  • -w（设置工作目录）：进容器直接是这个目录，免得每次都要切目录
  • -p（指定端口映射）：服务器端口 : 容器内端口，这里我设置了两个端口，5000的是服务要用到的不必理会，41258:22 将服务器41258，映射到容器内的22端口，记住41258这个端口

4. 安装

5.修改SSH配置文件

6. 重启ssh服务

二、Pycharm部分

不同版本的pycharm有些差异，但是大体上操作差不多，笔者版本是2024.1.5

1. 连接容器

步骤1：File-> Settings(快捷键：ctrl+alt+S)-> Project: -> Python Interpreter-> Add Interpreter-> On SSH..

步骤2：输入账号、密码

步骤3：指定docker上的python环境（这里我用的是conda），点击后pycharm会自动查找docker上的conda并列出已有的虚拟环境（也可选择新建虚拟环境）

步骤4：指定映射目录

三、注意

1. ssh软件包的坑

openssh-server、openssh-client 需要python环境，会自动下载python，并且对python版本有要求！

切换容器python版本可能导致ssh不可用

解决：使用Poetry、Miniconda之类的虚拟环境
